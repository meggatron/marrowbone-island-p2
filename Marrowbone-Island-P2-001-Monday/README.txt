Marrowbone Island – Day 1

Project Summary:
This is the starting version of our Python 2 course game, Marrowbone Island.
It’s a text-based adventure where you move between locations and make choices.
We’ll be building on this throughout the course.

Folder Structure:
main.py          - Starts the game
assets/intro.txt - Intro text shown at the start
assets/log.txt   - (Optional) Record of events
assets/map.jpg   - Map image of the island
game/locations.py- Location functions
game/player.py   - Player information

How to Play:
Run main.py to start the game.
